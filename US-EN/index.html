Python

Shell Scripting

Overall

To get the application / server performance monitoring software AppDynamics to work, we need to install agents in each of our servers to have it talk with the UI. However, there are a lot of Docker containers running on different servers and deploying an agent to each of the servers could be painful. That's why I wrote this script. The script reads in a config file which contains physical addresses of all servers and name of the application that is running in each Docker container. The script then generates an agent based on the application name from the config file and deploys that agent file into that container. This script has helped a lot on deploying agents to applications since the Docker containers run in different physical servers after each daily build.

This is partly what I did during my internship at Verizon in summer 2017.
